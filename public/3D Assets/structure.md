# Structure.md (public/3D Assets)

## Overview
This directory contains 3D GLB models used for vehicle rendering and world debris in the navigation interface.

## Subfolders
- **Textures** - Image maps (color, normal, metallic) for the 3D models.

## Files
- **ambulance.glb**, **police.glb**, **firetruck.glb** - Emergency vehicles.
- **sedan.glb**, **suv.glb**, **hatchback-sports.glb** - Personal vehicles.
- **truck.glb**, **delivery.glb**, **van.glb** - Commercial vehicles.
- **kart-oo*.glb** - Special avatar karts.
- **debris-*.glb** - Scenery/collision objects (axles, bumpers, plates).
- **wheel-*.glb** - Independent wheel models for modular vehicle assembly.
