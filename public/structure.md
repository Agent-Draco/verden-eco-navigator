# Structure.md (public)

## Overview
This directory contains static assets and environment-related configurations for the single-page application.

## Subfolders
- **3D Assets** - GLB/GLTF models and textures for vehicle rendering.
- **hdri** - High dynamic range environmental backgrounds.
- **Reference videos for AI** - Visual references for application behavior and styling.

## Files
- **favicon.ico**, **favicon.png** - Browser tab icon.
- **healthz** - Health check endpoint for monitoring.
- **placeholder.svg** - Generic SVG placeholder for missing content.
- **robots.txt** - Search engine indexing configuration.
