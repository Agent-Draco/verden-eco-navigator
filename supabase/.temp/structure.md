# Structure.md (supabase/.temp)

## Overview
This hidden directory is managed by the Supabase CLI and stores temporary state, versioning info, and connection parameters for the local development environment.

## Files
- **cli-latest**, **gotrue-version**, **postgres-version**, **rest-version**, **storage-version** - Specific tool versioning metadata.
- **project-ref** - Unique identifier for the shared or local Supabase project.
- **pooler-url** - Connection string for the Supabase connection pooler.
