# Structure.md (supabase)

## Overview
This directory contains the local Supabase configuration, environment settings, and connection details.

## Subfolders
- **.temp** - Temporary project state and version files managed by the Supabase CLI.

## Files
- **config.toml** - (If present) The main Supabase configuration file.
- **project-ref**, **postgres-version** - Metadata files for the linked Supabase project.
