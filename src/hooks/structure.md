# Structure.md (src/hooks)

## Overview
This directory contains custom React hooks used for shared logic, state synchronization, and map navigation utilities.

## Files
- **use-ecomoov.ts** - Hook for tracking eco-movement challenges and scoring.
- **use-mobile.tsx** - Reusable media query hook to detect small-screen/mobile devices.
- **use-toast.ts** - Facade hook for triggering Shadcn/Sonner toast notifications.
- **useGeoNavigation.ts** - Complex hook for managing real-time GPS tracking and navigation state.
- **structure.md** - This overview of custom hooks.
