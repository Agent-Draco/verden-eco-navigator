# Structure.md (src/pages)

## Overview
This directory contains React page components mapped to the application's URL routes.

## Files
- **Home.tsx** - Dashboard/Feed view.
- **Navigation.tsx** - Destination search and route selection.
- **NavigationScreen.tsx** - Active turn-by-turn navigation experience.
- **EcoMoov.tsx**, **EcoMoovGroup.tsx** - Sustainability challenges and group rewards.
- **Customize.tsx** - Vehicle and avatar personalization screen.
- **Profile.tsx**, **Wallet.tsx**, **Credits.tsx** - User settings, stats, and transaction history.
- **Login.tsx**, **Signup.tsx**, **ForgotPassword.tsx** - Authentication flow views.
- **Onboarding.tsx**, **Splash.tsx** - First-run experience and loading stages.
- **PrivacyPolicy.tsx**, **TermsAndConditions.tsx** - Compliance and legal documentation pages.
- **TripSummary.tsx** - Report on carbon savings and credits earned after a trip.
- **VehicleSetup.tsx** - Initial vehicle configuration wizard.
- **Resources.tsx**, **FuelEfficient.tsx** - Educational content and efficiency guides.
- **RouteComparison.tsx** - Side-by-side view for selecting the greenest path.
- **Index.tsx** - Root-level home fallback or landing logic.
- **NotFound.tsx** - Generic 404 error page.
