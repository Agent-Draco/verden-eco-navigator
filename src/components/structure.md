# Structure.md (src/components)

## Overview
This directory contains the application's reusable React components, divided into general UI primitives and specific product features.

## Subfolders
- **ui** - Generic low-level UI components (accessible primitives like buttons, inputs, dialogs).
- **verden** - Specific feature components for the Verden project (map, navigation bars, vehicles).

## Files
- **NavLink.tsx** - Reusable navigation link with active state handling.
- **theme-provider.tsx** - Context provider for color schemes (light/dark/standard).
- **structure.md** - This overview of the folder's structure and components.
