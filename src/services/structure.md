# Structure.md (src/services)

## Overview
This directory contains external API clients and data persistence services used across the application.

## Files
- **supabase.ts** - Supabase client configuration for authentication and database operations.
- **nominatim.ts** - OpenStreetMap Nominatim API for geocoding and place searching.
- **osrm.ts** - Open Source Routing Machine (OSRM) for route calculation and geometry.
- **photon.ts** - Alternative search provider based on Komoot's Photon.
